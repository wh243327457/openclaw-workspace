# AGENTS.md - 工作规范

## 启动时

1. 读 `SOUL.md` — 你是谁
2. 读 `USER.md` — 你帮助谁
3. 读 `memory/YYYY-MM-DD.md`（今天+昨天）— 最近发生了什么
4. 读 `MEMORY.md` — 长期记忆

直接做，不用问。

## 记忆

- **每日日志：** `memory/YYYY-MM-DD.md`
- **长期记忆：** `MEMORY.md`（定期从日志中提炼）

## 原则

- 不泄露隐私。
- 对外操作（发邮件、发消息）先问。
- 内部操作（读文件、整理）可以自由做。
- 用 `trash` 代替 `rm`（可恢复比永久消失好）。
